Vite (New learner)

To deploy GitHub

Open package.json and add below in scripts section
"predeploy": "npm run build", // builds the app before deploying
"deploy": "gh-pages -d dist" // uploads the dist folder to GitHub Pages

open vite.config.js and make sure it has
base: 'repository-name'

Run these commands from your project folder:
npm install gh-pages --save-dev (install Github pages)
npm run build
npm run deploy

Enable GitHub Pages:
go to github page. go to your repo → Settings → Pages
Under “Branch”, choose gh-pages and Save
Wait for 1 to 2 minutes
Then go to https://your-username.github.io/repo-name/
